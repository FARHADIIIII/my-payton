این برنامه با استفاده از کد پایتون نوشته شده است که با کلیک بر روی دکمه جستجو در ایات قرآنی میتوان با نوشتن نام میوه آیات قرآنی که درانها نام میوه آمده است همراه با ترجمه را مشاهده کرد. در قسمت خواص میوه ها نیز با انتخاب نام هر میوه خواص و فواید ان میوه نمایش داده میشود. برای اجرای برنامه فایل main  را اجرا کنید و توجه داشته باشید که برای اجرا باید کتابخانه های   Tkinter ، SQLite3  و PIL  نصب شوند.
